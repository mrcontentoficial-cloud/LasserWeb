# Clínica Electrónica Lásser — Sitio web

Sitio web oficial de **Clínica Electrónica Lásser** — Mantenimiento Electrónico Industrial.

Reparación de electrónica industrial a nivel componente (variadores VFD, servo drives, PLC, HMI, fuentes y tarjetas) en Irapuato y todo el Bajío.

## Archivos
- `index.html` — sitio completo en un solo archivo (HTML, CSS y JS en línea).
- `logo-lasser.png` / `img-lasser/` — logotipo e imágenes de la empresa.

## Publicar con GitHub Pages
1. En GitHub: **Settings → Pages**.
2. En *Source*, elige la rama `main` y la carpeta `/ (root)`.
3. Guarda. El sitio quedará en `https://mrcontentoficial-cloud.github.io/LasserWeb/`.
